# Cyber Defense System

A Secure Authentication System built using Flask and Python.

## Features
- User Registration
- Secure Login System
- Password Hashing
- OTP Verification via Email
- Forgot Password Functionality
- Account Lock after Multiple Failed Attempts
- Security Logging
- SQLite Database Integration

## Technologies Used
- Python
- Flask
- SQLite
- Flask-Mail
- HTML/CSS

## Future Enhancements
- Password Strength Meter
- Face Authentication
- Admin Dashboard
- Two-Factor Authentication (2FA)
- Security Analytics Dashboard
